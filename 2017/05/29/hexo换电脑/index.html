1. 源代码上云端

2. 在新电脑上配置hexo

将代码clone下来，master上是部署的博客，hexo分支上是我们的源代码，切到源代码分支

3. 继续愉快的写博客吧

4. 不成功？

1、服务提示 no index.html

检查thmes文件是否缺失，重新配置下主题，再次编译

2、 重新部署hexo不需要hexo init

如果运行了hexo init也没关系，重新配置 _config.yml文件
