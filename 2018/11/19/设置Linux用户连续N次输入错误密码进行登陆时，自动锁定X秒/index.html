备注：

1、此处使用的是 pam_tally2 模块，如果不支持 pam_tally2 模块可以使用 pam_tally 模块。另外，不同的pam版本，设置可能有所不同，具体使用方法，可以参照相关模块的使用规则。

3、可以使用 pam_tally2 -r -u username 命令，手动清除某用户记录次数。
